# Security Policy

## Reporting Security Issues

**Please DO NOT report security vulnerabilities through public GitHub issues.**

Instead, please report them via:
- Email: innergclaw@gmail.com
- Private GitHub Security Advisory

Please include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if any)

We will respond within 48 hours and work to resolve issues quickly.

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |

## Security Measures

This tool helps protect against:
- Hardcoded credentials
- Malicious dependencies
- Code obfuscation
- Suspicious network calls
- Destructive operations

**Note:** This scanner provides best-effort detection. Always review code manually.

## Best Practices

1. Keep this tool updated
2. Run scans before every skill installation
3. Report false positives to improve detection
4. Share knowledge with the community
